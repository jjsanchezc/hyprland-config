# Hyprland dots for Arch

Personal dotfiles for an Arch Linux system running the Hyprland Wayland compositor. Includes a dynamic theming system via matugen, a custom AGS shell, and configurations for a full desktop environment.

---

## Directory Overview

### `hyprland/`
Core Wayland compositor configuration.

- **hyprland.conf** — Main config: keybindings (Super key), window rules, workspace assignments, startup apps, and NVIDIA env vars.
- **monitors.conf** — Display setup (excluded from git, machine-specific).
- **workspaces.conf** — Workspace-to-monitor assignments (excluded from git).
- **colors.conf** — Border/accent colors. Overwritten by matugen on wallpaper change.
- **hyprlock.conf** — Lock screen: clock, date, user avatar, password input.
- **hypridle.conf** — Idle timeouts: screen dim → lock → suspend.
- **hyprpaper.conf** — Wallpaper daemon configuration.
- **Wallpapers/** — Stored wallpaper images.

### `ags/`
[AGS](https://github.com/Aylur/ags) (Astal GTK Shell) — custom GTK4 desktop widgets.

- **Bar.tsx** — Status bar/panel.
- **ControlCenter.tsx** — Slide-out panel: audio, network, bluetooth, battery, brightness, power profiles (asusctl), and media player (MPRIS).
- **Notifications.tsx** — Notification popup display.
- **style.scss / _colors.scss** — Styling. `_colors.scss` is overwritten by matugen.
- **hypridle-profiles.json** — Idle timeout profiles (configurable from the control center).

### `waybar/`
[Waybar](https://github.com/Alexays/Waybar) — top panel.

- **config.jsonc** — Module layout: workspaces (left), clock (center), CPU/RAM/network/audio/battery/tray (right). Clicking modules opens relevant controls.
- **style.css** — Styling, generated by matugen.

### `matugen/`
[Matugen](https://github.com/InioX/matugen) — generates a color palette from the current wallpaper and propagates it to all apps.

- **config.toml** — Defines template input/output paths and post-hooks (e.g., reload waybar, restart AGS, apply GTK theme).
- **templates/** — `.in` template files for: Hyprland colors, Hyprlock, Waybar CSS, AGS SCSS, Kitty colors, Rofi theme, Tmux colors, GTK4 CSS.

Activated via `wallpaper-set <path>`.

### `kitty/`
[Kitty](https://sw.kovidgoyal.net/kitty/) — primary terminal emulator.

- **kitty.conf** — JetBrainsMono Nerd Font (15px), 95% background opacity, block cursor.
- **colors.conf** — Generated by matugen.
- **minimal-dark.conf / mocha.conf** — Fallback/alternative color themes.

### `foot/`
[Foot](https://codeberg.org/dnkl/foot) — lightweight secondary terminal.

- **foot.ini** — Same font as kitty, 95% opacity, custom dark colorscheme, no scrollbar.

### `rofi/`
[Rofi](https://github.com/lbonn/rofi) — application launcher and menu.

- **config.rasi** — `drun` mode, single column, 7 visible entries, custom colors from matugen. `Ctrl+J/K` for navigation.

### `mako/`
[Mako](https://github.com/emersion/mako) — Wayland notification daemon.

- **config** — Minimal configuration, relies mostly on defaults.

### `wlogout/`
[Wlogout](https://github.com/ArtsyMacaw/wlogout) — power menu overlay.

- **layout** — Buttons: lock (hyprlock), logout, suspend, reboot, shutdown.
- **style.css** — Custom styling.
- **icons/** — Button icons.

### `swayosd/`
[SwayOSD](https://github.com/ErikReider/SwayOSD) — on-screen display for volume/brightness changes.

- **style.css** — Minimal OSD styling.

### `tmux/`
[Tmux](https://github.com/tmux/tmux) — terminal multiplexer.

- **tmux.conf** — Symlink to `~/.config/tmux/tmux.conf`.
- **colors.conf** — Generated by matugen.

### `nvim/`
[Neovim](https://neovim.io/) — text editor, configured with [NvChad](https://nvchad.com/).

- **init.lua** — Bootstraps Lazy.nvim + NvChad.
- **lua/** — Custom options, keymaps, autocmds, and plugin configs.
- **lazy-lock.json** — Plugin lock file for reproducible installs.

### `zsh/`
[Zsh](https://www.zsh.org/) shell configuration.

- **.zshrc** — Powerlevel10k prompt, fzf integration, zoxide, history settings, aliases (`ls`→eza, `vim`→nvim, `grep`→ripgrep), lf directory-change integration, and custom `Ctrl+J/K` history search.
- **.zshenv** — Environment variables: GTK/icon/cursor themes, QT theming, editor, terminal, locale, and `$PATH` additions (`~/.local/bin`, `~/.cargo/bin`).
- **.zprofile** — Login shell setup.
- **.p10k.zsh** — Powerlevel10k theme config.

### `lf/`
[lf](https://github.com/gokcehan/lf) — terminal file manager.

- **lfrc** — Vim-like keybindings, image preview via sixel, custom `open`/`mkdir`/`touch` commands.
- **preview** — Preview script using bat, eza, and file-type detection.
- **cleaner** — Cleanup script for sixel previews.
- **lfrun** — Wrapper that enables image previews.

### `scripts/`
Custom shell scripts, installed to `~/.local/bin/`.

| Script | Description |
|--------|-------------|
| `wallpaper-set` | Applies a wallpaper via hyprpaper, triggers matugen theming, caches path to `~/.cache/current-wallpaper` |
| `wallpaper-picker` | Rofi-based wallpaper browser with recursive folder navigation |
| `power-saver` | Monitors upower events; on battery sets 40% brightness + 60Hz, on AC sets 70% + 120Hz |
| `power-menu` | Rofi power menu (lock, suspend, logout, reboot, shutdown) |
| `battery-alert` | Sends notifications at 20% (low), 5% (critical), 100% (full) |
| `full-capture` | Screenshots the entire screen with grim, copies to clipboard, saves with timestamp |
| `region-capture` | Interactive region screenshot using slurp + wayfreeze for freeze-before-capture |
| `network-menu` | Rofi network management interface |
| `ags-watch` | Watches AGS source files (`.ts`, `.tsx`, `.scss`) and restarts AGS on changes |
| `pk-waybar` | Kills and restarts Waybar |
| `libthemes` | GTK theme switcher using symlinks |

### `systemd/`
Systemd user units.

- **battery-alert.service** — Runs the `battery-alert` script at login.
- **battery-alert.timer** — Periodic timer that fires `battery-alert` repeatedly.

### `themes/`
Visual theme assets.

- **.themes/jqnMono/** — Primary GTK theme.
- **.icons/Papirus-Light/** — Icon theme.
- **.config/Kvantum/** — Qt5/Qt6 theming (Fluent-round).

### `fonts/`
Font files and fontconfig rules installed to `~/.fonts` / `~/.config/fontconfig/`.

### `default-apps/`
- **.config/mimeapps.list** — Default application associations for file types.

### `nixos/` *(if on NixOS)*
Declarative system configuration (not used on Arch, kept for reference).

- **flake.nix** — Nixpkgs + home-manager inputs.
- **configuration.nix** — System: boot, networking, Hyprland, zsh, timezone, user.
- **home.nix** — User packages and dotfile symlinks via home-manager.
- **modules/** — Modular configs: packages, NVIDIA, fonts, services, dev tools.

## Theming

Colors are generated dynamically from the active wallpaper using **matugen**. Running `wallpaper-set <image>` will:

1. Set the wallpaper via hyprpaper.
2. Run matugen, which renders all templates and writes color files to the relevant config locations.
3. Post-hooks reload Waybar, restart AGS, and apply GTK/Qt themes.

A static fallback palette (`hyprland/colors.conf`, `ags/_colors.scss`) is used before any wallpaper is set.

## Startup Applications

Launched via `exec-once` in `hyprland.conf`:

| App | Role |
|-----|------|
| `swayosd-server` | OSD daemon for volume/brightness popups |
| `power-saver` | Battery-aware brightness and refresh rate management |
| `hypr-monitor-manager` | Auto display setup |
| `ags-watch` | AGS widget system with hot-reload |
| `hyprpaper` | Wallpaper daemon |
| `hypridle` | Idle manager (dim → lock → suspend) |
| `waybar` | Status bar |
| `kitty` (tmux) | Terminal session on workspace 1 |
| `kitty` (nvim) | Editor session on workspace 3 |

## Key Dependencies

- **WM**: Hyprland
- **Shell**: Zsh + Powerlevel10k
- **Terminal**: Kitty (primary), Foot (secondary)
- **Editor**: Neovim + NvChad
- **Widgets**: AGS (Astal GTK Shell)
- **Bar**: Waybar
- **Launcher**: Rofi (wayland fork)
- **Notifications**: Mako
- **Theming**: Matugen
- **Screenshots**: grim, slurp, wl-clipboard
- **Audio**: PipeWire + pamixer
- **Brightness**: brightnessctl
- **File manager**: Nautilus (GUI), lf (TUI)
- **Multiplexer**: Tmux
- **Tools**: fzf, zoxide, eza, bat, ripgrep, cliphist, lazygit
- **Font**: JetBrainsMono Nerd Font
